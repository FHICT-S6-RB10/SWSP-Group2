# Group 2 Services

## Run all containers using docker-compose
```
docker-compose pull && docker-compose up -d
```

## See all containers
```
docker ps
```

## See all images
```
docker images
```

## See all networks
```
docker network ls
```

## Stop all containers
```
docker-compose stop
```

## Delete all containers and networks
```
docker-compose down
```
